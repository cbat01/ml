#利用sklearn实现文本聚类

- 首先就是提取特征词，方法有卡方、tfidf等
- 然后将文本向量化，取到3000个特征词，如果文本中包含特征词，则把权重值置为tfidf值。
- 利用sklearn中的kmeans方法进行聚类。
- 由于此次语料本来就是分好类的可以打完标进行检验
## 结果

- 0 党政机关和社会组织
- 1 学校
- 2 文化体育
- 3 医院社保
- 4 水利环境和公共设施
